INSTALLATION
================================================================================

1. Clone repository
2. Set up web server (Apache/Nginx)
3. Create MySQL database
4. Run setup.php in browser
5. Follow 3-step wizard:
   - Step 1: Database credentials
   - Step 2: Admin user creation
   - Step 3: Complete!

Development Server:
  cd public
  php -S localhost:8000
